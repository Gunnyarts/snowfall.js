snowfall.js

A pure JavaScript/CSS snowfall animation

Snowfall on an image:

You can simply add snowfall.js to an image by targeting the class of the image container in the init-function.
See code examples below.

The CSS:


.snowfall {
  position: relative;
  overflow: hidden;
}

The HTML:


<div class="snowfall img-container">
  <img src="../link-to/image.jpg">
</div>

The JS:


snowDrop.init({bindto:".snowfall"})
